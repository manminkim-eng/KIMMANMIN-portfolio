## 🚀 사용 방법

### 1. 로컬에서 바로 확인
`index.html`을 더블클릭하거나 브라우저에 드래그하면 바로 포트폴리오가 열립니다.

### 2. 깃허브 페이지 업로드 (권장)
현재 운영 중이신 `manminkim-eng.github.io/KIMMANMIN/` 저장소에 업로드하면 웹으로 공개됩니다:

1. 깃허브 저장소에 접속: https://github.com/manminkim-eng/KIMMANMIN
2. 기존 `index.html`을 이 파일로 교체
3. `images/` 폴더를 통째로 업로드
4. 5~10분 뒤 https://manminkim-eng.github.io/KIMMANMIN/ 에서 바로 확인 가능

### 3. 웹호스팅 업로드
`index.html`과 `images/` 폴더를 그대로 FTP·호스팅 서비스에 업로드하시면 됩니다.

## 🔧 수정 가이드

### 텍스트·정보 수정
`index.html`을 메모장·VS Code 등 텍스트 편집기로 열어 원하는 문구를 수정하세요.

### 연락처 변경
파일 하단 `<!-- ============ CONTACT / FOOTER ============ -->` 섹션에서 수정

### 색상 변경
파일 상단 `:root { ... }` 블록의 `--accent`, `--navy` 등 값을 변경

### 프로젝트 추가/삭제
- 갤러리: `<div class="gallery-grid" id="galleryGrid">` 안에 `<div class="gallery-item">` 블록을 복사·수정
- 대표 프로젝트: `<div class="featured-grid">` 안에 `<div class="featured-project">` 블록을 복사·수정
